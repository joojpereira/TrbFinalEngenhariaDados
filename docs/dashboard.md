# Dashboard - Apache Superset

## Visao Geral

O dashboard foi construido no Apache Superset consumindo dados da camada Gold.

## KPIs

| KPI | Descricao |
|---|---|
| Total de Pedidos | Numero total de pedidos no periodo |
| Receita Total | Soma do valor total dos pedidos |
| Ticket Medio | Receita dividida pelo numero de pedidos |
| Clientes Ativos | Clientes com pelo menos 1 pedido |

## Metricas

| Metrica | Descricao |
|---|---|
| Pedidos por Mes | Evolucao mensal do volume de pedidos |
| Top 10 Produtos | Produtos com maior receita gerada |

## Como Acessar

1. Acesse http://localhost:8088
2. Login: admin / admin123
3. Va em Dashboards > Northwind Dashboard

## Importar Dashboard

O dashboard exportado esta em superset/dashboard_northwind.zip.
Acesse Dashboards > Import Dashboard e selecione o arquivo.
