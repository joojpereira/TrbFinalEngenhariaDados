# Trabalho Final - Engenharia de Dados

## Bem-vindo

Esta documentação apresenta a estrutura do projeto desenvolvido para a disciplina de Engenharia de Dados.

## Objetivos

- Construir um pipeline de dados
- Implementar arquitetura Medalhão
- Transformar dados utilizando Apache Spark
- Disponibilizar um modelo dimensional
- Criar um dashboard para análise

## Visão Geral

O projeto é composto por etapas de ingestão, armazenamento, transformação e disponibilização dos dados para consumo analítico.
